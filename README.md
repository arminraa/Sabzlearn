# Sabzlearn Clone

A modern clone of the Sabzlearn educational platform built with Next.js. This project recreates the user interface and user experience of the original website while implementing the frontend from scratch and developing a custom backend architecture.

> This project is created for educational and portfolio purposes. The codebase was written independently and is not copied from the original website.

## Demo

🌐 Live Demo: https://sabzlearn-gules.vercel.app/

*For accessing course category page please
enter /course-cat/slug


## Features

### Frontend

* Responsive modern UI
* Dark / Light mode support
* Home page
* Courses page
* Single course page
* Login page
* Mobile-friendly design
* Reusable component architecture
* Interactive sliders and carousels

### Backend (In Progress)

* User authentication
* Course management
* Database integration
* API routes
* Admin dashboard

## Tech Stack

## Installation

Clone the repository:

```bash
git clone https://github.com/arminraa/Sabzlearn
cd Sabzlearn
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
